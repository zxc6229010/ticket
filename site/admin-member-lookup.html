會員 / 帳號查詢中心

使用單一欄位快速查詢會員帳號、基本資料、後台角色與校園驗證資訊。 平台端可查全部資料；校方 / 組織方僅能查看自己權限範圍。
登入帳號：- 查詢範圍：-
可查詢欄位： email / uid / 學號 / 姓名 / schoolId / orgId / verifyKey / role / 驗證學校

查詢條件

輸入任一關鍵字後即可搜尋符合條件的會員或帳號資料。
查詢規則
1. 平台端可查全部帳號與會員。
2. 校方 / 組織方只可查自己範圍。
3. 查詢採模糊比對。
4. 結果會顯示角色、驗證資訊、基本欄位。
5. 後續可再擴充活動報名狀態、最近活動資訊。
